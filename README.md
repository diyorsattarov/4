## About The Project
This project is centered around the implementation of a robust, distributed system as a key learning experience. At its core, it leverages the Boost.Beast library to establish a websocket connection that efficiently serves data. This data is then processed and relayed by an Express server, forming a crucial bridge in our system architecture.

The front end of the project is powered by React, which is responsible for displaying the data received from the server. This architecture not only demonstrates the power and flexibility of distributed systems but also serves as a practical learning tool for understanding how different technologies can be integrated to create a cohesive and functional application.

The primary objective of this project is educational, providing a hands-on experience with distributed systems and showcasing the integration of various technologies like Boost.Beast, Express, and React.
